# ProofBench v2 data

The evaluation uses only `proofbench_v2.csv`: 60 proof problems, split evenly
between Basic and Advanced. Its SHA-256 digest is
`aa8b813dbd4068137e3d165e5da228f6e0e1cc85a91c37883e1791b954e43af0`.

Columns:

- `Problem ID`
- `Problem`
- `Solution`
- `Grading guidelines`
- `Category`
- `Level`
- `Short Answer`
- `Source`

The data originates from Google DeepMind's IMO-ProofBench release in
`google-deepmind/superhuman`. The dataset is CC BY 4.0 and the original source
code is Apache-2.0.
